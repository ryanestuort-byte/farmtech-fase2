# FarmTech Solutions - Fase 2

## Integrantes
- Ryan pablo correa de paula

## Projeto
Sistema de irrigação inteligente com ESP32.

## Componentes
- ESP32
- 3 botões (NPK)
- DHT22 (umidade)
- LDR (pH)
- Relé (bomba)

## Funcionamento
A bomba liga quando:
- Umidade < 50%
- Pelo menos 2 nutrientes ativos

Caso contrário, permanece desligada.

## Objetivo
Simular uma solução de IoT para agricultura, evitando desperdício de água.
